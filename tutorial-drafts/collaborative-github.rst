速识Github
--------------
**Github** 是一个基于 **Git** （一个开源的分布式版本控制系统）的版本控制和协作的代码托管平台，因为只支持git 作为唯一的版本库格式进行托管，因此得名Github。利用Github

Github协作开发
---------------
Github支持多人协作开发的三种合作模式：

- fork
开发者 fork 自己生成一个独立的分支，跟主分支完全独立，pull代码后，项目维护者可根据代码质量决定是否merge代码
- organization
- collaborator
开发者在这三种模式下的权限依次升高,
